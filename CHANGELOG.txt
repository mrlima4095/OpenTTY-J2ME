[ _Commands_ ]

- `su` - charge your user to root
- `sudo <command>` - run as superuser
- `import <lib>` - import a lib 
- `about` - show info about a lib
- `log <option>` - log system
- `logcat` - show logs in console
- `x11 <service>` - X.Org Server CLI
