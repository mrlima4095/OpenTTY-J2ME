About

OpenTTY é um terminal portátil focado em dispositivos com recursos limitados. Este mirror hospeda artefatos de release com ênfase em simplicidade, verificabilidade e disponibilidade.

Interface inspirada no estilo clássico do site da linguagem Lua (cabeçalho azul, layout limpo, tipografia simples), sem reprodução de marcas registradas.
